关于(about Canvax)

Canvax是什么

Canvax主要解决两个问题，把绘图元素抽象为虚拟的对象树，然后给这些抽象的对象赋予事件侦听的能力

比如： 

    var circle = new Circle({...});
    circle.on("click" , function(){
        //do something ...
    });
