tiger

用path 绘制 的tiger，有模有样的...... 绘制这个tiger的代码不多，所以我就把全部代码贴下面了，首先是照例要把对应用到的module加载进来，这里以kissy下为例

{% highlight js %} //kissy下加载所需模块 KISSY.use("demo/tiger/tigerData,canvax/Canvax ,canvax/display/Stage , canvax/shape/Path ,node" , function( S ,tigerData ,Canvax, Stage ,Path ){ //照例获取页面的容器dom，不要传id哦。canvax不负责帮你getElementById var box = S.one("#canvasTest"); var width = box.width(); var height = box.height(); var canvax = new Canvax({ id : "canvax", el : box }) var stage = new Stage({ id : "tiger", context:{ width : width, height: height } }); //canvax环境准备好，stage也准备好了后，就开始绘制根据数据获取tiger了 //在这里就开始托管stage也可以的亲。 S.each(tigerData , function( Data , i ){ //因为前面4个元素，是绘图的x,y,width,height不是真正的path数据 if(i<=3){ return; } //好了，开始真正的绘制tiger的每个元素，每根胡子，每块肌肉 var map = new Path({ context : { x:200, y:200, path :Data.path , lineWidth : Data.stroke-width, fillStyle:Data.fill, strokeStyle: Data.stroke=="none" ? null : Data.stroke } }); stage.addChild(map); }); //好了，stage已经准备完毕了。那么接下来就是把这个stage托管给canvax了。。。 //ps:其实，这个托管，在哪里添加都可以，each 前面的那个注释有说过... canvax.addChild(stage); }); {% endhighlight %}
